"Ceci est le début de ma page réalisée en Html. Tout cela grâce à la" Wild Code School.

Voici les différentes étapes du challenge:

  • 1. La première étape a été de configurer l'adresse e-mail ainsi que l'ID GitHub.

    Commandes: "git config --global user.email" et "git config -- global user.name"


  • 2. Création et préparation du dépôt local. Commande: git init


  • 3. Connexion du dépôt local à GitHub

    Commande: git remote add origin "https://github.com/jeanbaptiste-leduc/web"


  • 4. Ajout des fichiers au dépôt local Commande: git add .


  • 5. Validation des modifications Commande: git commit -m "First commit"


  • 6. Envoi des modifications vers GitHub Commande: git push -u origin master
